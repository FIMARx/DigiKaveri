Remote Support Guide

Follow these instructions to prepare your device for remote support. Remote connection is a secure way to get help without anyone needing to visit in person.

Computer

How to Start Remote Support on PC

We use TeamViewer, the world's most popular and secure remote support software.

Download the App

Click the button below to download the small utility program.

Download TeamViewer QS

Version: Windows (QuickSupport)

Want to download a different version or visit the official site?

TeamViewer Official Download Page
1

Open the downloaded file

Click the TeamViewerQS.exe file at the bottom of your browser or in your Downloads folder.

2

Allow the app to run

If Windows asks "Do you want to allow this app to make changes?", answer Yes.

3

Provide ID and Password

A window will open showing an ID and Password. Provide these to your DigiKaveri assistant over the phone.

Download the App

Click the button below to download the small utility program.

Download TeamViewer (Mac)

Version: macOS (QuickSupport)

Want to download a different version or visit the official site?

TeamViewer Official Download Page
1

Open and launch

Click TeamViewerQS.dmg and launch the app. If macOS asks "Are you sure you want to open?", answer Open.

2

Grant permissions

macOS requires remote management permissions. In the app, click "Review System Access" and allow DigiKaveri: Accessibility and Screen Recording.

3

Provide ID and Password

A window will open showing an ID and Password. Provide these to your DigiKaveri assistant over the phone.

Security Notice: The connection is one-time only. Once you close the program, we cannot access your computer without a new ID and password, which change every time.

Android Guide

1

Download the app

Click the button below to download the QuickSupport app from the Google Play Store.

Download on: Google Play
2

Open and allow permissions

Open the app and allow the necessary permissions (such as notifications and screen sharing) so we can help you remotely.

3

Tell the ID code

Read the 9-digit ID code shown on the app screen to your DigiKaveri helper over the phone.

The connection is protected. We can only see your device during this call while the app is open.

iPhone & iPad Guide

Download on: App Store
2

Open and allow permissions

Open the app and allow the necessary permissions (such as notifications and screen sharing) so we can help you remotely.

Tell the ID code

Read the 9-digit ID code shown on the app screen to your DigiKaveri helper over the phone.

The connection is protected. We can only see your device during this call while the app is open.

Note: Some devices might ask to install an Add-On for full remote control. Please accept the installation if prompted by the app.
